# Collective Intelligence (CI²)

CI² is an independent research system for measuring hedge fund reputation using news, social data, workplace reviews, and brand materials.

## Pipeline

Collector → Enricher → Mention Tagger → Scoring → Time Series → Dashboards

## Components

- QWASS — News reputation
- SCUM — Social sentiment
- WERK — Workplace reputation
- DORIAN — Brand vs reputation alignment
